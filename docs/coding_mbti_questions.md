# 코딩MBTI 질문지

## 코딩MBTI 4가지 축

| 축 | 유형 A | 유형 B | 측정 내용 |
|----|--------|--------|-----------|
| 1 | T (생각형) | P (실행형) | 문제를 어떻게 풀까요? |
| 2 | S (계획형) | F (자유형) | 어떻게 정리할까요? |
| 3 | I (혼자형) | C (함께형) | 어떻게 배울까요? |
| 4 | E (안정형) | N (신기술형) | 어떤 도구를 좋아할까요? |

---

## 점수 체계

각 선택지는 강도에 따라 점수 부여:
- **A**: 첫 번째 유형 강함 (+2점)
- **B**: 첫 번째 유형 약함 (+1점)
- **C**: 두 번째 유형 약함 (+1점)
- **D**: 두 번째 유형 강함 (+2점)

---

## 축 1: 문제 풀이 방법 (T 생각형 vs P 실행형)

### Q1. 레고나 퍼즐을 할 때 나는?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 설명서를 처음부터 끝까지 완벽히 읽고 이해한 뒤 시작해요 | T +2 |
| B | 설명서를 먼저 훑어보고 큰 그림을 파악한 뒤 시작해요 | T +1 |
| C | 설명서는 막힐 때만 보고 일단 만들어봐요 | P +1 |
| D | 설명서 없이 내 방식대로 조립하는 게 더 재밌어요 | P +2 |

### Q2. 수학 문제나 퀴즈를 풀 때 나는?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 공식과 원리를 완전히 이해한 후에 문제를 풀어요 | T +2 |
| B | 예제 풀이를 분석하고 방법을 익힌 뒤 풀어요 | T +1 |
| C | 일단 풀어보고 틀리면 다른 방법을 시도해요 | P +1 |
| D | 감으로 먼저 답을 예측하고 여러 방법을 실험해요 | P +2 |

### Q3. 게임에서 새로운 스테이지를 만나면?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 공략을 찾아보고 완벽한 전략을 세운 뒤 도전해요 | T +2 |
| B | 먼저 패턴을 관찰하고 나만의 전략을 세워요 | T +1 |
| C | 몇 번 해보면서 감을 잡고 방법을 찾아가요 | P +1 |
| D | 일단 뛰어들어서 부딪히며 배우는 게 재밌어요 | P +2 |

### Q4. 뭔가 잘 안 될 때 나는?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 처음부터 차근차근 되짚으며 원인을 분석해요 | T +2 |
| B | 어디서 잘못됐는지 논리적으로 추론해봐요 | T +1 |
| C | 여기저기 바꿔보면서 해결책을 찾아요 | P +1 |
| D | 아예 다른 방법으로 새롭게 시도해봐요 | P +2 |

## 축 2: 정리 방법 (S 계획형 vs F 자유형)

### Q5. 새로운 숙제나 프로젝트를 시작할 때?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 상세한 계획표를 만들고 체크리스트로 관리해요 | S +2 |
| B | 할 일 목록을 정리하고 순서를 정해요 | S +1 |
| C | 대략적인 방향만 잡고 하면서 조정해요 | F +1 |
| D | 계획 없이 바로 시작하고 흐름대로 진행해요 | F +2 |

### Q6. 파일이나 공부 자료를 정리할 때?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 폴더 구조와 이름 규칙을 정해서 체계적으로 관리해요 | S +2 |
| B | 주제나 날짜별로 분류해서 나름 정리해요 | S +1 |
| C | 크게 분류만 하고 세부 정리는 나중에 해요 | F +1 |
| D | 검색하면 되니까 정리에 시간 쓰지 않아요 | F +2 |

### Q7. 여행을 준비할 때 나는?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 시간대별 일정과 예약을 완벽하게 계획해요 | S +2 |
| B | 가볼 곳 리스트와 대략적인 동선을 준비해요 | S +1 |
| C | 숙소만 정하고 나머지는 현지에서 정해요 | F +1 |
| D | 무계획이 계획! 발길 닿는 대로 다녀요 | F +2 |

### Q8. 내 방이나 책상 상태는?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 모든 물건에 자리가 있고 항상 정돈되어 있어요 | S +2 |
| B | 가끔 어지럽지만 주기적으로 깔끔하게 정리해요 | S +1 |
| C | 좀 어지럽지만 내가 찾기엔 편한 상태예요 | F +1 |
| D | 창의적인 혼돈! 어지러워도 상관없어요 | F +2 |

## 축 3: 공부 방법 (I 혼자형 vs C 함께형)

### Q9. 새로운 걸 배울 때 나는?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 혼자 책이나 영상으로 깊이 파고드는 게 좋아요 | I +2 |
| B | 혼자 먼저 공부하고 필요하면 질문해요 | I +1 |
| C | 선생님이나 전문가에게 직접 배우는 게 좋아요 | C +1 |
| D | 친구들과 함께 토론하며 배우는 게 제일 재밌어요 | C +2 |

### Q10. 모르는 문제가 생기면?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 끝까지 혼자 고민하고 검색해서 해결해요 | I +2 |
| B | 한참 고민해보고 안 되면 찾아보거나 물어봐요 | I +1 |
| C | 좀 고민하다가 아는 사람에게 물어봐요 | C +1 |
| D | 바로 친구나 선생님과 같이 해결해요 | C +2 |

### Q11. 공부할 때 가장 집중이 잘 되는 환경은?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 아무도 없는 조용한 내 방이 최고예요 | I +2 |
| B | 도서관처럼 조용하고 각자 집중하는 곳이 좋아요 | I +1 |
| C | 카페처럼 적당한 소음과 사람이 있으면 좋아요 | C +1 |
| D | 친구들과 함께 있으면 더 의욕이 생겨요 | C +2 |

### Q12. 팀 프로젝트를 할 때 나는?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 역할 분담 후 각자 완성해서 합치는 게 효율적이에요 | I +2 |
| B | 내 파트는 혼자 집중해서 하고 가끔 공유해요 | I +1 |
| C | 정기적으로 만나서 진행 상황을 나누며 해요 | C +1 |
| D | 처음부터 끝까지 다 같이 모여서 하는 게 좋아요 | C +2 |

## 축 4: 도구 선택 (E 안정형 vs N 신기술형)

### Q13. 새로운 앱이나 프로그램을 고를 때?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 오래되고 많은 사람이 쓰는 검증된 걸 선택해요 | E +2 |
| B | 평점과 리뷰를 확인하고 안정적인 걸 골라요 | E +1 |
| C | 새로 나온 거라도 기능이 좋으면 써봐요 | N +1 |
| D | 새로운 앱을 제일 먼저 써보는 게 즐거워요 | N +2 |

### Q14. 발표 자료를 만들 때 사용하는 도구는?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 파워포인트처럼 모두가 쓰는 표준 도구가 편해요 | E +2 |
| B | 익숙한 도구를 쓰되 가끔 새 기능을 써봐요 | E +1 |
| C | Canva 같은 새로운 도구를 적극적으로 활용해요 | N +1 |
| D | AI 도구 등 최신 기술을 먼저 시도해봐요 | N +2 |

### Q15. 휴대폰 앱 업데이트 알림이 오면?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 후기를 충분히 확인하고 안정화되면 업데이트해요 | E +2 |
| B | 며칠 기다렸다가 문제없으면 업데이트해요 | E +1 |
| C | 새 기능이 궁금해서 곧바로 업데이트해요 | N +1 |
| D | 베타 버전도 신청해서 먼저 써보는 편이에요 | N +2 |

### Q16. 유튜브나 SNS에서 주로 보는 콘텐츠는?
| 선택지 | 내용 | 점수 |
|--------|------|------|
| A | 오랫동안 검증된 유명하고 유용한 콘텐츠 | E +2 |
| B | 많은 사람들이 추천하는 인기 콘텐츠 | E +1 |
| C | 요즘 뜨는 새로운 트렌드나 최신 정보 | N +1 |
| D | 아직 유명하지 않은 새로운 분야 탐색 | N +2 |

## 점수 계산 방법

### 1단계: 각 축별 점수 합산
| 축 | 질문 | 점수 범위 |
|----|------|-----------|
| T vs P | Q1~Q4 | T: 0~8 / P: 0~8 |
| S vs F | Q5~Q8 | S: 0~8 / F: 0~8 |
| I vs C | Q9~Q12 | I: 0~8 / C: 0~8 |
| E vs N | Q13~Q16 | E: 0~8 / N: 0~8 |

### 2단계: 유형 결정
각 축에서 점수가 높은 유형 선택
- 동점일 경우: 중립 표시 또는 추가 질문

### 예시
Q1=B(T+1), Q2=C(P+1), Q3=B(T+1), Q4=A(T+2) → T:4, P:1 → **T**

## 16가지 코딩MBTI 유형

| 유형 | 별명 | 특징 | 어울리는 분야 |
|------|------|------|---------------|
| TSIE | 튼튼한 설계자 | 체계적이고 꼼꼼하게 설계해요 | 서버 개발, 시스템 설계 |
| TSIN | 새로운 발견가 | 새로운 방법을 연구하고 찾아요 | 인공지능, 연구 개발 |
| TSCE | 든든한 리더 | 팀과 함께 큰 프로젝트를 이끌어요 | 대기업 개발팀 리더 |
| TSCN | 기술 선생님 | 새로운 기술을 팀에 알려줘요 | 기술 리더, 멘토 |
| TFIE | 만능 개발자 | 여러 가지를 다 잘해요 | 풀스택 개발 |
| TFIN | 스타트업 개척자 | 혼자서도 척척 만들어내요 | 1인 개발, 스타트업 |
| TFCE | 협력 전문가 | 팀원들과 유연하게 협력해요 | 팀 프로젝트, 애자일 |
| TFCN | 혁신 주도자 | 팀과 함께 새로운 걸 만들어요 | 혁신팀, R&D |
| PSIE | 실용적인 만들이 | 실제로 쓸 수 있는 걸 만들어요 | 앱 개발, 서비스 개발 |
| PSIN | 화면 디자이너 | 예쁘고 편한 화면을 만들어요 | 프론트엔드, UI/UX |
| PSCE | 협업 개발자 | 팀과 함께 서버를 만들어요 | 백엔드 팀 개발 |
| PSCN | 배포 전문가 | 프로그램을 세상에 내보내요 | DevOps, 인프라 |
| PFIE | 자유 개발자 | 혼자서 여러 프로젝트를 해요 | 프리랜서, 다양한 외주 |
| PFIN | 인디 크리에이터 | 나만의 작품을 만들어요 | 인디 게임, 개인 앱 |
| PFCE | 팀 조율자 | 팀이 잘 돌아가게 도와줘요 | PM, 스크럼마스터 |
| PFCN | 스타트업 멤버 | 빠르게 움직이는 팀에서 활약해요 | 초기 스타트업 팀원 |
